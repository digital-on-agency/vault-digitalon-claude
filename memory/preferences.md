# Preferences

## Comunicazione con i clienti
Queste preferenze si applicano SOLO quando si scrive contenuto destinato a un cliente
(messaggi, email, documenti). Non si applicano alla comunicazione interna o alle note operative.

### Lingua
- Italiano per tutti i clienti (salvo indicazioni specifiche nel CLAUDE.md del cliente)

### Tono per tipo di contenuto
- **Messaggio** (WhatsApp, chat): informale, diretto, conciso
- **Email**: formale ma non rigido, professionale
- **Preventivo/Documento**: formale, strutturato, professionale

### Personalizzazione per cliente
- Le preferenze specifiche per ogni cliente (tono, lingua, termini da usare/evitare)
  vanno definite nel file `clients/[cliente]/CLAUDE.md`
- Le preferenze specifiche sovrascrivono quelle generali qui definite
