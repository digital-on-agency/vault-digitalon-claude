**Data**: 2025-11-28
**Partecipanti**: Guido (Digital On), Francesco Leo
**Durata**: ~40 min

## Riepilogo
Definita soluzione definitiva per integrazione disegni: AutoCAD Web + export PDF + upload nel gestionale. Nessuna integrazione nativa replicabile in cloud.

## Punti chiave
- Integrazione nativa Excel↔AutoCAD non replicabile in ambiente cloud
- Alternativa valutata: AutoCAD Web — incluso nella licenza AutoCAD LT di Francesco (~€460/anno)
- Interfaccia AutoCAD Web simile a desktop — Francesco può lavorare da qualsiasi dispositivo
- Workflow concordato: AutoCAD Web → esporta PDF → carica nel gestionale nel campo specifico del disegno
- Google Drive proposto come storage centralizzato PDF (no occupazione spazio locale)
- Francesco ha espresso preoccupazione sicurezza Drive vs NAS → rassicurato sulla stabilità Google
- Digital On fornisce inizialmente il proprio spazio Drive; eventualmente il cliente attiva abbonamento

## Decisioni prese
[BLOCCATO] AutoCAD Web + export PDF + upload gestionale — soluzione definitiva, nessuna integrazione nativa
- Digital On implementa collegamento nel gestionale che apre AutoCAD Web direttamente
- Francesco responsabile dell'export PDF e del caricamento nel sistema
- Google Drive come storage centralizzato dei PDF disegni
